    # POS System Backend

    Sebagai referensi dan juga titik repo untuk BackEnd, didalamnya terletak beberapa code untuk Projek ke-1.

    Fitur:

        REST API untuk Manajemen Projek

        Logika Proses Transaksi

        Schema Database MYSQL

    How to run HTR:

        Clone repo dengan hati - hati

        npm install

        Import database.sql ke MySQL

        npm start

    API Endpoints:

        GET /products - Nge-fetch Daftar Item yang ada didalam database

        POST /checkout - Proses Transaksi yang akan terjadi.
